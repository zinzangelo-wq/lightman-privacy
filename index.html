Datenschutzerklärung für die Lightman Prompt Engine App

Letzte Aktualisierung: Januar 2026

Die Lightman Prompt Engine App verarbeitet keine personenbezogenen Daten. Es werden keinerlei Informationen über Nutzer gesammelt, gespeichert, geteilt oder analysiert.

1. Erhobene Daten

Es werden keinerlei personenbezogene Daten erhoben, darunter:

  • Keine Namen oder Kontaktinformationen
  • Keine Geräte- oder Standortdaten
  • Keine Tracking- oder Analyse-Daten
  • Keine Werbung und keine externen Tracker
  • Keine Nutzerprofile oder Identifikatoren

2. Verarbeitung von Texteingaben

Vom Nutzer eingegebene Texte werden ausschließlich lokal auf dem Gerät verarbeitet. Es findet keine Speicherung im Backend und keine Weitergabe an Dritte statt.

3. Regel-Synchronisierung

Die App besitzt eine Funktion zur Aktualisierung von Optimierungsregeln. Dabei werden ausschließlich Regeltexte synchronisiert, die vom App-Besitzer erstellt wurden.
Diese Regeln enthalten keine Nutzerdaten.

4. Rechte der Nutzer

Da keine personenbezogenen Daten verarbeitet werden, ergeben sich keine Ansprüche auf Löschung, Export oder Berichtigung.
